# Blackpad

Blackpad is a minimal WinUI Markdown editor built around a distraction-free black writing surface and Instrument Serif typography.

## Features

- Borderless black editor window
- Markdown source editing
- Native rendered preview for headings, bullets, quotes, code blocks, bold, italic, and inline code
- Editor-only, split-preview, and preview-only modes
- Full settings page
- Supports `.md`, `.markdown`, and `.txt` files
- Bundles the official Instrument Serif regular and italic font files under the SIL Open Font License

## Shortcuts

- `F11`: Toggle fullscreen
- `Ctrl+E`: Open settings
- `Ctrl+O`: Open `.md`, `.markdown`, or `.txt`
- `Ctrl+S`: Save
- `Ctrl+Shift+S`: Save as `.md` or `.txt`

## Build And Run

```powershell
dotnet build
dotnet run
```

The app uses WinUI 3 and the Windows App SDK.
